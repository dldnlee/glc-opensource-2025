# GLC Open Source 2025에 기여하기

기여해 주셔서 감사합니다! 아래 단계를 따라주세요:

1. **저장소 포크 및 클론**  
   저장소를 포크하고 로컬에 클론하세요:

```bash
git clone https://github.com/your-username/glc-opensource-2025.git
```

2. **브랜치 생성**  
   새 브랜치를 만드세요:

```bash
git checkout -b feature-name
```

3. **변경 및 커밋**  
   변경 후 커밋하세요:

```bash
git commit -m "설명"
```

4. **푸시 및 풀 리퀘스트**  
   변경 사항을 푸시하고 풀 리퀘스트를 만드세요:

```bash
git push origin feature-name
```

## 문제 보고 및 질문

이슈를 열거나 논의를 시작하세요.

감사합니다!
